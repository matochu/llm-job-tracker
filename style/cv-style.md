# CV Style

## Default Language

- CV/resume: English.
- Cover letter: English.

## House Style

- Do not add a `Target Role` section.
- `Career Expectations` should be a meta professional preference, not a cover letter.
- Do not over-name the target company in generic CV sections.
- Avoid long dash punctuation in prose.
- Avoid LLM filler such as:
  - "what drives me is"
  - "force multiplier"
  - "at home in"
  - "holds up under real-world scale"
  - "leaving things better than I found them"
- Use plain evidence and real outcomes.

## CV Positioning

- Summary should show relevance in the first 10 seconds.
- Skills should be ordered by vacancy importance.
- Experience bullets should support vacancy keywords with proof, not only list technologies.
- Keep earlier career compressed enough not to dilute recent relevance.
- Never invent stack, scale, metrics, or domain experience.

## Review Rubric

Score CV fit using the resolved profile plus these shared review dimensions:

- role target clarity
- company/domain fit
- vacancy keyword coverage
- evidence strength
- level calibration
- career expectations fit
- work mode/logistics
- risk profile
- differentiation
- narrative consistency
- ATS/readability
- interview readiness
